## 🎨 创建示例

### 建筑生成
```javascript
// 随机生成 20 个建筑
for (let i = 0; i < 20; i++) {
    engine.createBuilding(
        10,
        Math.random() * 20 + 10,
        10,
        new THREE.Vector3(
            (Math.random() - 0.5) * 150,
            0,
            (Math.random() - 0.5) * 150
        )
    );
}
```

### 角色创建
```javascript
const character = engine.createCharacter(
    new THREE.Vector3(0, 0, 0)
);
```

### 粒子效果
```javascript
const particles = engine.createParticleSystem(
    1000,
    0x00ff00,
    new THREE.Vector3(0, 5, 0)
);
```
