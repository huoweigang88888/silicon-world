## 🎯 使用指南

### 第一步：创建 Agent

1. 点击左侧 **"➕ 创建 Agent"** 按钮
2. 填写信息：
   - **名字**: 例如 "硅基助手"
   - **控制者地址**: 0x 开头的钱包地址
   - **人格类型**: 选择友好/专业/创意等
3. 点击 **"✨ 创建"**

### 第二步：查看 Agent

1. 在左侧列表点击 Agent
2. 右侧显示详细信息：
   - ID、控制者、创建时间
   - 人格配置、激活状态
3. 可以加载记忆或删除 Agent

### 第三步：管理记忆

1. 点击 **"💭 加载记忆"**
2. 查看该 Agent 的所有记忆
3. 在搜索框输入关键词搜索
4. 切换选项卡查看不同内容

## 🏗️ 技术架构

### 前端技术栈
- **纯 HTML5** - 无需框架
- **原生 JavaScript** - 无构建步骤
- **CSS3** - 渐变、动画、响应式
- **Fetch API** - 异步请求

### 文件结构
```
web/dashboard/
└── index.html          # 单文件应用（23KB）
```

### API 集成
- **基础地址**: `http://localhost:8000`
- **健康检查**: `/health`
- **Agent API**: `/api/v1/agents`
- **记忆 API**: `/api/v1/agents/{id}/memories`

## 🔧 自定义

### 修改配色
编辑 `index.html` 中的 CSS：
```css
/* 主色调 */
background: linear-gradient(135deg, #667eea 0%, #764ba2 100%);

/* 改成你的颜色 */
background: linear-gradient(135deg, #你的颜色 0%, #你的颜色 100%);
```

### 添加功能
在 `<script>` 标签中添加 JavaScript 函数。

### 扩展 API 调用
使用 `fetch()` 添加新的 API 请求。

## ⚠️ 注意事项

1. **API 必须先运行**

2. **跨域问题**
   - 当前 API 已启用 CORS
   - 如果部署到不同域名，需要配置 CORS

3. **浏览器兼容**
   - 需要现代浏览器（Chrome/Firefox/Safari/Edge）
