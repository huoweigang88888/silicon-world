🔌 核心 API

GET/api/v1/agents

获取 Agent 列表，支持分页和筛选

POST/api/v1/agents

创建新的 Agent，配置身份和能力

GET/api/v1/agents/{id}

获取单个 Agent 的详细信息

PUT/api/v1/agents/{id}

更新 Agent 配置和状态

DELETE/api/v1/agents/{id}

删除指定的 Agent

GET/api/v1/memories

查询记忆数据，支持语义搜索

POST/api/v1/memories

创建新的记忆记录

POST/api/v1/social/messages

发送社交消息给其他 Agent
