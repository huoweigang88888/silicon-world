## 🚀 快速开始

### 第一步：创建钱包 (2 分钟)

1. 访问 **Dashboard** 页面
2. 点击 **"钱包"** 菜单
3. 点击 **"创建钱包"**
4. 设置密码（至少 8 位）
5. **重要：备份助记词！**

![创建钱包](assets/create-wallet.png)

⚠️ **安全提示**:
- 助记词只能看到一次
- 不要截图或发送给任何人
- 建议手抄在纸上保存
- 丢失助记词 = 丢失资产

### 第二步：获取测试 ETH (3 分钟)

1. 复制您的钱包地址
2. 访问水龙头网站:
   - https://goerlifaucet.com/
   - https://faucets.chain.link/goerli
3. 粘贴地址
4. 点击领取
5. 等待 1-2 分钟到账

![获取测试 ETH](assets/get-test-eth.png)

### 第三步：铸造您的第一个 NFT (5 分钟)

1. 访问 **NFT 市场** 页面
2. 点击 **"铸造"** 菜单
3. 填写 NFT 信息:
   - 类型（土地/建筑/艺术品等）
   - 名称
   - 描述
4. 上传作品图片
5. 点击 **"铸造 NFT"**
6. 确认交易
7. 等待确认（约 15-30 秒）

![铸造 NFT](assets/mint-nft.png)

✅ **恭喜！您成功铸造了第一个 NFT！**

## 💱 NFT 交易

### 购买 NFT

1. 访问 **交易** 页面
2. 选择 **"购买"** 标签
3. 浏览可购买的 NFT
4. 点击 **"立即购买"**
5. 确认支付
6. 等待交易确认

### 出售 NFT

1. 访问 **交易** 页面
2. 选择 **"出售"** 标签
3. 选择要出售的 NFT
4. 设置价格（ETH）
5. 点击 **"上架出售"**
6. 授权合约（首次需要）
7. 确认上架

## 🐛 常见问题

### Q: 钱包密码忘记了怎么办？

A: 很遗憾，密码无法恢复。您需要：
1. 删除钱包
2. 重新创建
3. 使用助记词恢复（如果有备份）

**建议**: 使用密码管理器保存密码

### Q: 交易一直确认中怎么办？

A: 可能原因：
1. 网络拥堵 - 稍等片刻
2. Gas 费用太低 - 重新发送交易
3. 余额不足 - 充值测试 ETH

### Q: NFT 铸造失败怎么办？

A: 检查以下几点：
1. 余额是否足够（包含 Gas 费）
2. 网络连接是否正常
3. 浏览器控制台是否有错误
4. 刷新页面重试

### Q: 如何查看我的交易记录？

A: 三种方式：
1. 钱包页面 - 查看交易历史
2. Etherscan - 输入钱包地址查询
3. 我的 NFT 页面 - 查看持有 NFT

### Q: 可以取消已上架的 NFT 吗？

A: 可以！
1. 访问交易页面
2. 选择"我的交易"标签
3. 找到要取消的 NFT
4. 点击"取消上架"

## 🔐 安全提示

### 保护您的资产

1. ✅ **备份助记词** - 手抄在纸上，不要截图
2. ✅ **保护密码** - 使用强密码，不要告诉他人
3. ✅ **验证网址** - 确保访问正确的网站
4. ✅ **小心钓鱼** - 不点击可疑链接
5. ✅ **定期检查** - 查看交易记录

### 常见骗局

❌ **不要相信**:
- "官方客服"索要助记词
- "空投"要求连接钱包
- "免费 ETH"要求私钥
- 不明链接要求签名

✅ **记住**:
- 官方永远不会索要助记词
- 不要随意签名不明消息
- 大额交易先小额测试
