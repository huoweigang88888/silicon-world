## 🏆 核心亮点

### 技术亮点

1. **双层架构设计**
   - 编排层 (Zoe) + 执行层 (Agent)
   - 清晰的职责分离

2. **完整的社交系统**
   - 好友/关注/消息/群组/通知/屏蔽
   - 20+ 个社交 API 端点

3. **实时通信能力**
   - WebSocket 实时推送
   - 消息/通知即时到达

4. **灵活的 Agent 系统**
   - 支持原生和外部 Agent
   - 5 种预设模板一键导入

5. **三层记忆系统**
   - 短期记忆 (临时)
   - 长期记忆 (持久)

2. **群组管理** - 踢人/禁言/退出
3. **屏蔽系统** - 拉黑骚扰者

5. **模板系统** - 快速接入常见 Agent

### 体验亮点

1. **美观的 Dashboard** - 紫色渐变设计
