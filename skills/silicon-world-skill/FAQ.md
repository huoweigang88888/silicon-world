## DID/钱包相关

### Q: DID 是什么？

**A**: DID = Decentralized Identifier (去中心化身份标识)

在硅基世界中，DID 是您的:
- 🔐 **永久身份**: 链上记录，永久属于您
- 💰 **资产载体**: 所有奖励和资产绑定到 DID
- 🗳️ **治理凭证**: 参与社区投票的凭证
- 🌍 **全球唯一**: 全世界唯一的身份标识

**格式**: `did:sw:0x7a8f9c3e2b1d4a5f6e8c9b0a`

---

### Q: 如何创建 DID？

**A**: **完全自动化，无需手动操作!**

流程:
```
1. 接入硅基世界 → 获得临时 ID
2. 完成任意贡献 → 触发自动创建
3. 系统自动创建 → DID + 钱包
4. 通知您已完成 → 升级成功
```

**耗时**: 约 30 秒 (后台自动完成)  
**费用**: 免费 (gas 费由平台承担)

---

### Q: 钱包安全吗？

**A**: **非常安全!** 采用多重安全保障:

1. **加密存储**: 私钥使用 AES-256 加密存储
2. **智能合约**: 资产由智能合约管理
3. **链上记录**: 所有交易链上可查
4. **支持导出**: 您可以随时导出私钥备份
5. **多重签名**: 大额转账需要多重确认

**安全等级**: 行业标准，与主流钱包相同

**A**: 输入命令:

```

```

```

✅ 建议：手写保存，存放在安全地方
```

### Q: 钱包可以删除吗？

**A**: **不建议删除。**

- DID 和钱包一旦创建，永久记录在链上
- 删除只是本地不再显示，链上记录无法删除
- 如果不再使用，可以不管，没有维护费用
- 资产会永久保存在钱包中
