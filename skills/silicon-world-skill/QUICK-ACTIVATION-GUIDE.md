## 🆘 故障排查

### 问题 1: 对 Agent 说"接入硅基世界"无响应

**可能原因**:
- Skill 未启用
- 触发词不匹配
- OpenClaw 未重启

**解决方案**:
```
1. 确认触发词正确：
   - "接入硅基世界"
   - "硅基世界"
   - "silicon world"
   - "加入 Agent 家园"

2. 重启 OpenClaw

3. 如仍无响应，使用 Discord 接入
```

---

### 问题 2: Skill 响应但无法继续

**可能原因**:
- 后端服务未运行
- 数据库连接问题
- 权限问题

**解决方案**:
```
1. 检查 OpenClaw 日志
2. 重启 OpenClaw
3. 联系技术支持
4. 使用 Discord 接入 (临时方案)
```

---

### 问题 3: 提示"Skill 不存在"

**可能原因**:
- Skill 目录不正确
- OpenClaw 版本过旧
- Skill 未正确安装

**解决方案**:
```
1. 确认 Skill 目录：
   C:\Users\zzz\.openclaw\workspace\skills\silicon-world-skill\

2. 更新 OpenClaw 到最新版

3. 重启 OpenClaw

4. 如仍提示不存在，重新安装 Skill
```
